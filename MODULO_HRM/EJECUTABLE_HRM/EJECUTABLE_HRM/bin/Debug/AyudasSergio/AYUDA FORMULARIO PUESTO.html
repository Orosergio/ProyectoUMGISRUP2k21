AYUDA FORMULARIO PUESTO
En este formulario de puesto el usuario podrá realizar lo siguiente:
Insertar: Al presionar este botón se habilitarán todos los campos excepto el campo
Código, en forma vacía para que el usuario pueda ingresar un nuevo registro.
Modificar: Al presionar este botón se habilitarán todos los campos excepto el
campo Código, del registro en el que se encuentre posicionado actualmente, y
podrá ser modificado, según lo quiera el usuario.
Eliminar: Al presionar este botón se mostrará un mensaje de advertencia al
usuario, si el usuario acepta, se eliminará el registro en el cual el usuario se
encuentre posicionado actualmente.
Guardar: Si el usuario desea guardar la inserción de un nuevo registro o la
modificación de uno ya existente, debe presionar este botón para que los datos
queden registrados en la base de datos.
Cancelar: Al presionar este botón se cancelará toda acción que se esté realizando
en el momento.
Refrescar: se actualizará nuevamente el formulario.
Consultar: se le mostrará una nueva ventana donde podrá consulares datos
específicos del formulario.
Botones de guía: podrá navegar entre los registros de la tabla que se le muestran.
Ayuda: se abrirá el archivo de ayuda sobre el formulario.
Salir: se cerrará el formulario.
Campo Código: El sistema generara un código automático que corresponda al numero del
registro que correspondería.
Campo Puesto: Campo que contendrá el nombre del puesto que se desea almacenar o
modificar.
Campo Descripción: Campo que contendrá una breve descripción sobre el puesto.
Campo Salario: Este campo contendrá el salario que este puesto ofrece.
Campo Estado: se debe seleccionar activo para que el campo se almacene en la base de
datos y este pueda ser visible, pueda ser utilizado en registros futuros.
